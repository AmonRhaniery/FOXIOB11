Federated Knowledge Extraction Framework

FOX is a framework that integrates the Linked Data Cloud and makes use of the diversity of NLP algorithms to extract RDF triples of high accuracy out of NL. In its current version, it integrates and merges the results of Named Entity Recognition tools. Keyword Extraction and Relation Extraction tools will be merged soon.


Documentation

The approache, datasets and evaluation as well as the FOX API are descriped in the documentation.


Framework

FOX is a framework and can be used programmatically, for more informations read the Java documentation.


Web API

FOX comes with a RESTful web service and can be used with FOX-Java or FOX-Python bindings.

To use the RESTful web service directly, read more about the parameter in the documentation or try out the demo.


Tools

FOX uses AGDISTIS, an Open Source Named Entity Disambiguation Framework able to link entities against every Linked Data Knowledge Base, to disambiguate entities against DBpedia. In contrast to lookup-based approaches, our framework can also detect resources that are not in DBpedia.

FOX relies on ensemble learning and combines the results of several NER tools. It utilizes the Waikato Environment for Knowledge Analysis (Weka) and a Multilayer Perceptron trained with the largest dataset (5) included in our evaluation.
